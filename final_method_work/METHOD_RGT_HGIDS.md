# RGT-HGIDS

**RGT-HGIDS** is the final paper-oriented method name for this project.

- English: **Redundancy-aware Gated Temporal Heterogeneous Graph Intrusion Detection System**
- Chinese: **冗余感知门控时序异构图入侵检测框架**
- Core pipeline: **TBB-RR + RGD-BiGRU-MCBG + ST-HGAN + EHA**
- Main implementation encoder: `semantic_encoder=rgd_bigru_mcbg`
- Main experiment script: `scripts/run_rgt_hgids_rigorous.sh`
